1. 意向形（表示意图或建议）
五段动词的意向形通过将动词的最后音节的“う”变为“お”段音，再加上“う”来构成。

行く（いく，iku） → 行こう（いこう，ikou） - 一起去吧
書く（かく，kaku） → 書こう（かこう，kakou） - 一起写吧
飲む（のむ，nomu） → 飲もう（のもう，nomou） - 一起喝吧
待つ（まつ，matsu） → 待とう（まとう，matou） - 一起等吧
2. 可能形（表示可能性）
五段动词的可能形通过将动词的最后音节的“う”变为“え”段音，再加上“る”来构成。

行く（いく，iku） → 行ける（いける，ikeru） - 能去
書く（かく，kaku） → 書ける（かける，kakeru） - 能写
飲む（のむ，nomu） → 飲める（のめる，nomeru） - 能喝
待つ（まつ，matsu） → 待てる（まてる，materu） - 能等
3. 命令形（表示命令或要求）
五段动词的命令形通过将动词的最后音节的“う”变为“え”段音。

行く（いく，iku） → 行け（いけ，ike） - 去！
書く（かく，kaku） → 書け（かけ，kake） - 写！
飲む（のむ，nomu） → 飲め（のめ，nome） - 喝！
待つ（まつ，matsu） → 待て（まて，mate） - 等！
二、一段动词（即“る”结尾的动词）
一段动词的末尾都是“る”音，变形规则相对简单。

1. 意向形
一段动词的意向形通过直接将“る”变为“よう”来构成。

食べる（たべる，taberu） → 食べよう（たべよう，tabeyou） - 一起吃吧
見る（みる，miru） → 見よう（みよう，miyou） - 一起看吧
寝る（ねる，neru） → 寝よう（ねよう，neyou） - 一起睡吧
2. 可能形
一段动词的可能形通过将“る”变为“られる”来构成。

食べる（たべる，taberu） → 食べられる（たべられる，taberareru） - 能吃
見る（みる，miru） → 見られる（みられる，mirareru） - 能看
寝る（ねる，neru） → 寝られる（ねられる，nerareru） - 能睡
3. 命令形
一段动词的命令形通过直接将“る”变为“ろ”来构成。

食べる（たべる，taberu） → 食べろ（たべろ，tabero） - 吃！
見る（みる，miru） → 見ろ（みろ，miro） - 看！
寝る（ねる，neru） → 寝ろ（ねろ，nero） - 睡！
三、不规则动词
不规则动词有一些特殊的变化规则，主要是する和来る这两个动词。

1. する（做）
意向形 → しよう（しよう，shiyou） - 一起做吧
可能形 → できる（できる，dekiru） - 能做
命令形 → しろ（しろ，shiro） - 做！
2. 来る（来）
意向形 → 来よう（こよう，koyou） - 一起来吧
可能形 → 来られる（こられる，korareru） - 能来
命令形 → 来い（こい，koi） - 来！
